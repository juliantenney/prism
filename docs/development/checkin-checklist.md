# PRISM Pre-Checkin Checklist

## Architecture & Continuity

- [ ] Relevant docs updated
- [ ] Significant decisions recorded
- [ ] Current-state snapshot updated
- [ ] No unresolved architectural contradictions introduced

## Runtime & Stability

- [ ] Runtime paths verified
- [ ] Manifest/config paths valid
- [ ] No broken references from file moves
- [ ] Smoke-check completed if needed

## Cleanup

- [ ] Legacy/experimental files reviewed
- [ ] No accidental dead code introduced
- [ ] Naming remains consistent
- [ ] Archive used instead of deletion where appropriate

## Strategic Direction

- [ ] Changes align with current consolidation priorities
- [ ] No uncontrolled feature expansion introduced

## Session Continuity

- [ ] Session handover prepared
- [ ] Next bounded task identified
- [ ] New-chat bootstrap prompt prepared if needed
- [ ] Sprint folder prepared/updated (`docs/development/sprints/YYYY-MM-DD-sprint-name/`)
