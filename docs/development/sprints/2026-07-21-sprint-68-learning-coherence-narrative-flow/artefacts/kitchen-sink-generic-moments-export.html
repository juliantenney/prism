Learner Renderer Kitchen Sink — Sprint 68 Type Coverage

57 minutes

Overview

Synthetic fixture exercising all thirteen vNext-supported material types across five archetype-matched activities (KS01–KS05).

Learning purpose

Validate renderer coverage, type-to-surface mapping, and safe fallback behaviour without modifying GAM or PRISM semantics.

Knowledge summary

Each activity maps to one archetype variant. Materials are minimal but schema-valid.

Learning outcomes

By the end of this lesson, you should be able to:

  1. Recognise static instructional material rendering. (LO-KS1)
  2. Complete table and text learner surfaces where composition applies. (LO-KS2)
