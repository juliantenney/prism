## Mission

Improve the **fidelity** with which instructional components already generated by **DLA** and **GAM** reach the **learner-facing workbook page**.

Sprint 38-G proved that richer instructional design can be produced using existing KM/LO structures and the Activity Component Model (ACM). The remaining quality gaps are **primarily realisation fidelity gaps** — not a need to redesign KM, LO, or ACM.

**Focus chain:**

```text
DLA  →  GAM  →  Workbook Page
         ↑
    (preserve instructional richness)
```

**Assumed valid (do not reopen):**

```text
KM  →  LO  →  ACM  →  DLA  →  GAM
```

---

## Programme question

> When DLA and GAM intentionally author instructional framing, guidance, worked reasoning, verification, reflection, and transition cues, **what prevents those decisions from reaching the learner unchanged** — and how can we fix that **without** schema expansion or structural contract regression?

---

## Primary success condition

A learner-facing workbook page **preserves** intentionally generated:

- instructional framing  
- guidance  
- worked reasoning  
- verification  
- reflection  
- transition prompts  

The system must **not** collapse rich instructional materials into task shells or spoiler closures.

**Hold condition (38-E / 38-F / 38-G gains):**

> Retain **V-01**, **V-05**, 38E-8/9 material types, 38G-3 ACM rows (DLA-WB-20/21, GAM-WB-21), and **Preservation PASS** (V-13 monitor).

**Secondary measure:**

> `EV-38H-AFTER` realisation trace shows **fewer fidelity losses** than `EV-38G-AFTER` on the same anchor, with **no regression** on structural holds.
