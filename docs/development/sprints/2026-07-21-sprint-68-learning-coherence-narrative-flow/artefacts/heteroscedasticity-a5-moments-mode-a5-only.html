Detection and Remedy Options

Detection

  • Residual plots provide visual evidence.
  • The Breusch–Pagan test provides a formal statistical check.

Remedies

  • Robust standard errors improve inference when variance is not constant.
  • Logarithmic transformations may reduce variability differences.
  • Improved model specification may address omitted influences.

Evaluation criteria:

  1. Suitability for available evidence.
  2. Support for reliable inference.
  3. Practical simplicity.

Research Scenarios

Scenario A

A researcher observes a clear funnel pattern in a residual plot from a household spending model.

Scenario B

A researcher has a large dataset but no visual diagnostics yet and wants formal evidence regarding changing variance.

Worked Judgement

In Scenario A, the residual plot already provides direct evidence of a widening pattern. Because the available evidence is visual and immediately observable, inspecting the residual plot is highly suitable. It is also simple to implement. However, visual inspection alone may not provide the same level of formal confirmation as a statistical test.

A Breusch–Pagan test could add diagnostic depth, but it requires an additional analytical step. The trade-off is therefore simplicity versus stronger diagnostic evidence.

If the goal is to improve the reliability of inference after heteroscedasticity has been identified, robust standard errors become particularly valuable. They do not explain why variance changes, but they help address the consequences for statistical inference.

Considering the evidence available, the need for reliable inference, and practical simplicity, the most appropriate immediate action in Scenario A is to begin with the residual plot evidence and then use robust standard errors if inference is the main concern. This judgement places greater weight on the observed pattern while recognising the limitation that visual evidence alone may not fully diagnose the issue.

Key Takeaways

Heteroscedasticity occurs when residual variance changes across observations rather than remaining constant. Residual plots provide a practical way to detect changes in variability, while statistical tests such as the Breusch–Pagan test can provide supporting evidence. The main concern for economists is that changing variance can affect standard errors and therefore the reliability of statistical inference. Common responses include robust standard errors, transformations, and improved model design. When interpreting regression results, focus on evidence, patterns, and the implications for research conclusions.
