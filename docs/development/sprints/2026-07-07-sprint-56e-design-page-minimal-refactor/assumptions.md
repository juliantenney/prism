# Assumptions

1. Upstream artefacts are authoritative.
2. Design Page can access DLA, GAM, Episode Plans, Learning Sequence, and Learning Outcomes for the run under test.
3. Full GAM material bodies must be copied into the emitted page artefact.
4. Activity IDs must be preserved.
5. Material IDs must be preserved and mapped consistently.
6. Design Page is a packaging/assembly stage, not a design/authoring stage.
7. Additional complexity should only be introduced when a concrete failed invariant requires it.
