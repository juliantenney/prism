How to think

Move from evidence to interpretation. The variable itself does not prove heteroscedasticity; the residual pattern provides the evidence.

See it modelled

Review the modelled economic example involving household income and expenditure.

Applied Economic Reasoning

Household Income and Expenditure

Evidence:

  • Lower-income households: residuals typically ±50.
  • Higher-income households: residuals typically ±400.

Reasoning:

  1. Compare residual spread across groups.
  2. Notice systematic increase in variability.
  3. Link the pattern to changing variance.
  4. Conclude that evidence is consistent with heteroscedasticity.
