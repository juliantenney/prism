\# Sprint 40 Validation Report



\## Purpose



This document records the validation activities undertaken during Sprint 40 and the evidence supporting adoption of the PRISM Educational Quality Framework.



The purpose of validation was to determine whether the framework is:



\* Coherent

\* Educationally meaningful

\* Promptable

\* Domain independent

\* Actionable

\* Capable of improving generated learning resources



\---



\# Framework Under Evaluation



The framework defines educational quality through the development of:



\* Understanding

\* Capability

\* Judgement

\* Independence



Supported by:



\* Metacognition

\* Learning Success

\* Progressive Independence

\* Authentic Learning Design



\---



\# Validation Criteria



The framework should:



1\. Explain educational quality.

2\. Generate recognisably different learning experiences.

3\. Be applicable across domains.

4\. Support practical implementation.

5\. Improve existing PRISM outputs.

6\. Remain comprehensible to learning designers.



\---



\# Validation Activity 1



\## Feedback Literacy Exemplar



\### Topic



Giving Effective Feedback on Academic Writing



\### Method



A learning resource was generated using framework-informed prompting.



The resulting resource was evaluated against the framework.



\### Findings



The generated journey demonstrated:



\* Understanding development

\* Capability development

\* Judgement development

\* Independence development

\* Metacognitive support



Prompt refinement significantly improved:



\* Visible learner progress

\* Progressive independence

\* Learning success indicators



\### Key Finding



Framework concepts successfully translated into generation instructions and became visible in the resulting learning experience.



\---



\# Validation Activity 2



\## Was Marx Right?



\### Topic



Evaluation of Marx's critique of capitalism.



\### Method



A second exemplar was generated using the same framework and prompting approach.



The topic was deliberately selected because it represented a contested intellectual question rather than a procedural or skills-based topic.



\### Findings



The generated resource demonstrated:



\* Conceptual understanding

\* Analysis of ideas

\* Comparison of competing perspectives

\* Evaluation of evidence

\* Position formation

\* Defence of conclusions



The learner journey naturally progressed through:



Observe

→ Explore

→ Analyse

→ Compare

→ Evaluate

→ Decide

→ Defend

→ Transfer



\### Key Finding



The framework successfully supported learning design within a complex humanities topic.



Domain independence was strongly demonstrated.



\---



\# Validation Activity 3



\## Existing PRISM Output Review



\### Resource



Inflation learning resource.



\### Method



An existing PRISM output was reviewed using the framework as an evaluation tool.



\### Findings



The resource was already educationally strong.



However, the framework identified opportunities for improvement through:



\* Richer understanding development

\* Stronger judgement development

\* Increased learner independence

\* More visible learning success

\* Greater metacognitive support



\### Key Finding



The framework provides a practical method for evaluating and improving existing outputs.



\---



\# Major Findings



\## Framework Coherence



All major educational concepts identified during Sprint 40 were successfully integrated into a single model.



No significant concepts remained without a place within the framework.



\---



\## Promptability



Framework concepts can be expressed through prompt language.



Changes in prompting produced observable changes in generated outputs.



This demonstrates that the framework is operationalisable.



\---



\## Domain Independence



The framework functioned successfully across:



\* Learning skills

\* Humanities

\* Disciplinary learning



This supports the conclusion that the framework operates at an appropriate level of abstraction.



\---



\## Educational Explanatory Power



The framework provides explanations for:



\* Activity selection

\* Journey design

\* Learner progression

\* Educational quality



This represents a significant improvement over activity-first approaches.



\---



\## Practical Utility



The framework can be used to:



\* Guide generation

\* Evaluate outputs

\* Improve existing resources

\* Inform future PRISM development



\---



\# Cognitive Activity Principle



A significant implementation insight emerged during validation.



Educational quality is primarily driven by cognitive activity rather than interface activity.



PRISM should therefore prioritise:



\* Thinking

\* Comparing

\* Evaluating

\* Reflecting

\* Deciding

\* Justifying



Interactivity should only be used when it meaningfully contributes to:



\* Understanding

\* Capability

\* Judgement

\* Independence



This principle is considered foundational for future implementation.



\---



\# Conclusions



Sprint 40 successfully produced an Educational Quality Framework that is:



\* Coherent

\* Portable

\* Promptable

\* Domain independent

\* Educationally meaningful

\* Actionable



The framework provides a foundation for future PRISM development by defining:



\* What educational quality means

\* How learner development occurs

\* How learning journeys should be designed

\* How generated outputs can be evaluated and improved



The framework represents a transition from generating learning resources to generating learner-development journeys.
