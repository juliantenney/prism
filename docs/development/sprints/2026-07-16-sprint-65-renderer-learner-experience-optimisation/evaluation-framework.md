# Sprint 65 — Evaluation Framework

Use this matrix for baseline (S65-BL-001) and before/after prototype/validation scoring. Prefer evidence and screenshots over aesthetic judgement alone.

**Do not** claim learning-outcome improvement without learner research. Prefer: presentation clarity · instructional legibility · task orientation · cognitive signalling · usability.

---

## Audit matrix

Score each representative page **before** and **after** on:

| Dimension | Question |
| --------- | -------- |
| Orientation | Can the learner tell what the session is and where to begin? |
| Action clarity | Is the immediate task obvious? |
| Output clarity | Is the required product obvious? |
| Cognitive mode | Is the intended kind of thinking obvious? |
| Progression | Is the relationship to previous and next work clear? |
| Material role | Can the learner tell how each material should be used? |
| Verification | Can the learner check and improve their work? |
| Transfer | Is application beyond the immediate example visible? |
| Density | Is important information prioritised over supporting detail? |
| Duplication | Is repeated information serving a clear purpose? |
| Accessibility | Is structure perceivable without visual styling alone? |
| Resilience | Does missing or unknown data fail safely? |

Suggested scale (document in each audit): **1** poor · **2** weak · **3** adequate · **4** strong · **5** excellent — with a one-line evidence note per cell.

---

## Design principle (scoring lens)

### Page level

1. What is this session about?  
2. What will I be able to do?  
3. How will the session progress?  
4. Where should I begin?  

### Activity level

1. What kind of thinking am I doing?  
2. Why am I doing this now?  
3. What must I do?  
4. What must I produce?  
5. What support should I use?  
6. How do I check my work?  
7. How does this transfer elsewhere?  

### Material level

1. Is this explanation, example, scenario, workspace, prompt or check?  
2. What role does it play in completing the activity?  
3. Should I read it, inspect it, complete it or use it to verify?
