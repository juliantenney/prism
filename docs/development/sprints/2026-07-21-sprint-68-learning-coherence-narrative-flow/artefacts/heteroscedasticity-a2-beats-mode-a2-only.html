Worked Residual Plot Interpretation

Interpreting a Residual Plot

Observation: Residuals form a widening funnel.

Analysis:

  1. The spread is narrow on the left.
  2. The spread increases as fitted values rise.
  3. Residual variance is changing.
  4. The pattern is systematic rather than random.

Judgement: The plot provides evidence consistent with heteroscedasticity.
