# Findings

- A minimal assembler-style Design Page output appears achievable in practice.
- Activity membership and ordering can be preserved from authoritative upstream inputs.
- Episode-plan attachment is straightforward when IDs are stable.
- Material fidelity remains the critical invariant and primary failure mode.
- Existing Design Page validation can produce false positives (for example, success flags while material bodies are summarized/replaced).
- The dominant challenge is specification and validation design, not baseline model capability.
