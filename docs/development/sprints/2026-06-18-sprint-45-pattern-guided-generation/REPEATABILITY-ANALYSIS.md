# Repeatability Analysis

**Date:** 2026-06-18  
**Type:** Methodological analysis — not a protocol, sprint plan, or implementation design  
**Purpose:** Determine what would have to be true for the Sprint 45.1 evaluation approach to become a repeatable research method

## Background

Sprint 45.1 answered its primary research question positively within scope: pattern-guided treatment improved 44-2 verdicts on four Photosynthesis pairs, held Strong on two Marx pairs, cleared target failure modes, and preserved ownership without mimicry flags.

That answer depended on a **multi-layer evaluation approach** — verdict-first contract scoring, secondary Detection Signals, failure-mode checks, ownership audit, anti-mimicry comparison, boundary declaration where needed, and convergent judgement for pair classification. The method **worked once** on six obligation-matched pairs in a single evaluator session ([`45-1-recommendation.md`](45-1-recommendation.md); [`METHODOLOGICAL-CONTRIBUTIONS-OF-SPRINT-45-1.md`](METHODOLOGICAL-CONTRIBUTIONS-OF-SPRINT-45-1.md)).

Repeatability is now the primary frontier because:

1. **45-1 recommendation gates further pattern injection** on evaluation standing — not on additional generation evidence alone.
2. **Generation claims require trustworthy observation** — without repeatable evaluation, intervention outcomes cannot scale as research findings.
3. **Repeatability was explicitly not measured** — no independent re-score, no second session, no replication variance ([`SPRINT-45-2-CHARTER.md`](SPRINT-45-2-CHARTER.md)).
4. **Sprint 44 established contract repeatability on frozen corpus**; Sprint 45.1 has not established equivalent repeatability for **intervention evaluation**.

*Interpretation:* Repeatability is the question of whether the 45-1 evaluation approach can become **standing research infrastructure**, not whether SP-02/SP-03 can influence bodies again on the same twelve artefacts.

## Judgement-Dependent Components

Assessment of repeatability risk per component.

### Contract scoring

| Factor | Assessment |
| ------ | ---------- |
| **Demonstrated** | All 12 bodies scored; verdicts drove all classifications |
| **Repeatability support** | Closed 44-2 text; Sprint 44 showed discrimination across 25+ materials; justifications cite §5.6/§5.8 bullets |
| **Repeatability risk** | Sprint 44 documented calibration splits (M2, M22, M23); TP-PS-A6 required boundary declaration before scoring |
| **Judgement dependence** | **Moderate** — tier assignment on strong/minimum boundary; ambiguous bodies need Layer 6 |

### Failure-mode identification

| Factor | Assessment |
| ------ | ---------- |
| **Demonstrated** | FM-04, FM-02, FM-03 recorded per body; paired delta on remediation |
| **Repeatability support** | Named FMs in 44-3 Secondary Index with detection signals; target FMs pre-specified per pattern |
| **Repeatability risk** | FM presence at Minimum tier (M12 + FM-04) requires judgement beyond tier alone |
| **Judgement dependence** | **Moderate** — present/absent is clearer than contract tier on boundary bodies |

### Ownership auditing

| Factor | Assessment |
| ------ | ---------- |
| **Demonstrated** | All 12 bodies; separability shown (TP-PS-A4 baseline: pass + Failed) |
| **Repeatability support** | Concrete checks — empty judgement cells, no pre-written transfer |
| **Repeatability risk** | Exemplar cell population vs modelling move (DT-PS) requires discrimination |
| **Judgement dependence** | **Moderate-high** on edge cases; **low** on clear violations (pre-filled judgement would be unambiguous) |

### Anti-mimicry analysis

| Factor | Assessment |
| ------ | ---------- |
| **Demonstrated** | All six pairs; mimicry suspect = no |
| **Repeatability support** | Corpus comparison structure (proximity, distance, suspect flag) |
| **Repeatability risk** | No automatic text-overlap metric; "domain-appropriate" is reviewer judgement; no positive mimicry case to calibrate |
| **Judgement dependence** | **High** — charter lists non-operationalisable mimicry as 45-2 failure condition |

### Boundary declarations

| Factor | Assessment |
| ------ | ---------- |
| **Demonstrated** | Once — TP-PS-A6 conjunctive §5.8 minimum |
| **Repeatability support** | Precedent enables scoring without global calibration reopening |
| **Repeatability risk** | Which declaration to apply may vary by evaluator; only one instance |
| **Judgement dependence** | **High** — declaration choice determines verdict on ambiguous bodies |

### Convergent judgement

| Factor | Assessment |
| ------ | ---------- |
| **Demonstrated** | All four Improvements met convergent bar; design rules pre-specified |
| **Repeatability support** | Explicit necessary vs corroborative vs disqualifying evidence ([`EVIDENCE-STANDARDS-ANALYSIS.md`](EVIDENCE-STANDARDS-ANALYSIS.md)) |
| **Repeatability risk** | Synthesis when layers conflict untested in-sample (no veto paths triggered) |
| **Judgement dependence** | **Moderate** — aggregation rules are design-specified; edge-case synthesis unproven |

## Potential Sources Of Drift

Where evaluator disagreement could emerge — from Sprint 44 calibration history, 45-1 design, and charter failure conditions.

| Drift source | Mechanism | 45-1 exposure |
| ------------ | --------- | ------------- |
| **Strong / Minimum boundary** | Contract tier on partial exemplar adequacy, word bands, context framing | DT-PS, TP pairs mostly clear deltas; M22 ambiguous |
| **M22-like calibration splits** | Conjunctive vs compensatory §5.8 minimum | TP-PS-A6 — declaration required |
| **Detection Signals vs verdict** | Signals suggest Strong; verdict Minimum/Failed | Not on treatments; baselines show separability |
| **FM at Minimum tier** | Minimum verdict + FM present (M12 shape) | DT-PS baselines |
| **Ownership: modelling vs completing** | Exemplar cells — scaffold or deliverable? | DT treatments; zero regressions |
| **Anti-mimicry threshold** | How much Strong-reference similarity is mimicry? | Not stress-tested |
| **Substantive vs cosmetic move** | Table fill vs exemplar reasoning; slot vs scaffolding | Reviewer notes; no disputed cases recorded |
| **Declaration selection** | Which boundary rule governs ambiguous body | Single instance |
| **Convergent synthesis under partial agreement** | Some layers pass, others borderline | Not observed |
| **Familiarity with 45-1 bodies** | Re-scoring same twelve vs new bodies | Re-scoring not done |

**Charter failure indicators:** Systematic verdict divergence = contract ambiguity problem; systematic signal/verdict divergence = two-layer model incoherent; non-operationalisable ownership/mimicry = method failure.

## Evidence Against Repeatability

| Finding | Why it weakens confidence |

| **Independent reviewers produce incompatible verdicts on same bodies** | Contract ambiguity dominates; protocol cannot fix |
| **Re-scoring twelve bodies yields different pair classifications** | 45-1 outcomes were session artefacts |
| **Signals and verdicts systematically diverge on new bodies** | Two-layer model incoherent |

| **Ownership/mimicry depend on irreducible intuition** | Layers not operationalisable |

## Strongest Current Hypothesis

**Evidence-supported hypothesis:**

> The Sprint 45.1 evaluation approach is **internally coherent and operable on a controlled six-pair sample**, but **repeatability is unproven**. The components most likely to repeat are **verdict-first contract scoring with pre-specified pair classification rules** and **binary FM/ownership checks on clear cases**. The components most likely to drift are **anti-mimicry judgement**, **boundary declaration selection**, and **substantive-vs-cosmetic discrimination on borderline bodies**. Repeatability requires independent re-application — especially re-scoring of the twelve 45-1 bodies and evaluation of new bodies — with convergence on pair classifications and operational disqualifier paths.

**Supported by:** High within-sample consistency; Sprint 44 contract discrimination precedent; design pre-specification; zero layer conflicts in 45-1.

**Not supported (hypothesis only):** That a written protocol will achieve inter-rater agreement — speculated in charter, not demonstrated.

## Conclusion

### Current state of confidence

| Dimension | Confidence level | Basis |
| --------- | ---------------- | ----- |
| **Operability (once)** | **High** | Six pairs complete; design criteria met; stack exercised |
| **Internal consistency** | **High** | Zero classification conflicts; layers aligned on all Improvements |
| **Repeatability (cross-session)** | **Low** — unmeasured | Single evaluator; no re-score |
| **Repeatability (cross-rater)** | **Low** — unmeasured | No inter-rater data on 45-1 bodies |
| **Repeatability (new bodies)** | **Low** — unmeasured | Twelve artefacts only |
| **Disqualifier path reliability** | **Unknown** | Designed but not triggered |
| **Standing research method** | **Not established** | 45-2 chartered to test |

### What would have to be true for repeatability

For the 45-1 evaluation approach to become a **repeatable research method**, the following would need to hold — stated as conditions, not as protocol design:

1. **Normative layer stable** — 44-2 verdicts can be applied compatibly by different reviewers on the same GAM bodies without reopening contracts.
2. **Classification rules sufficient** — Verdict ordering + FM/ownership gates + disqualifiers yield same pair classification when layers align.
3. **Judgement layers operationalisable** — Anti-mimicry, substantive-vs-cosmetic, and boundary declaration can be documented and applied with acceptable agreement.
4. **Disqualifier paths work under stress** — Ownership regression and mimicry flags function when borderline or positive cases appear.
5. **Generalisation beyond sample** — Method produces consistent conclusions on bodies not identical to the 45-1 twelve.
6. **Utility threshold met** — Agreement and cost support use as gate for further generation research.

**Demonstrated today:** Conditions 2 holds for one session on six pairs where layers did not conflict. Conditions 1, 3, 4, 5, 6 are **open**.

**Summary:** Repeatability is the **correct next frontier** because 45-1 proved the evaluation approach can work once; nothing yet proves it can work again. Confidence in **operability** is high; confidence in **repeatability** remains low until 45-2 or equivalent independent re-application provides evidence.
