# Core vs Extension Content Policy

## Definitions

- **Core embedded content:** required to complete required activities and outcomes.
- **Optional supporting resources:** useful but not required for baseline completion.
- **Further reading:** external depth for interested learners.
- **Extension tasks:** optional advanced/challenge work.
- **References:** provenance/citation resources.

## Rule of thumb

Core page content should support required activity completion.  
Additional depth should be externalised unless essential to target learning outcomes.
