## 1. What 38-H set out to do

Improve **fidelity** with which instructional components already generated by **DLA** and **GAM** reach the **learner-facing workbook page**.

**Focus chain:**

```text
DLA  →  GAM  →  Workbook Page
```

**Assumed valid (not redesigned):**

```text
KM  →  LO  →  ACM  →  DLA  →  GAM
```

**Programme question:** When DLA and GAM intentionally author framing, guidance, worked reasoning, verification, reflection, and transition cues, **what prevents those decisions from reaching the learner unchanged** — and how can that be fixed **without** schema expansion or structural contract regression?

**Programme position after 38-H:**

```text
38-C … 38-F   Workbook definition + structural contracts
38-G          Activity component quality (ACM) — SIGNIFICANT SUCCESS
38-H          Realisation fidelity — SUCCESS
38-I          Instructional episode model (pedagogical target state) — CHARTERED
```

The limiting question shifts from *“Can Prism preserve workbook structures?”* to *“What should excellent self-directed instructional episodes look like?”*
