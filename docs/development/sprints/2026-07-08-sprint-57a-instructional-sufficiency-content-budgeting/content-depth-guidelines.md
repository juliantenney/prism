# Content Depth Guidelines

## Principles

- Core materials should be sufficient to complete required activities.
- Not every topic requires textbook-level treatment inside core page content.
- Depth beyond core learning outcomes should usually be optional/supporting resource material.
- Richer is not automatically better if it displaces learner activity time.

## Evaluation lens

Depth decisions should balance:
- conceptual sufficiency,
- activity completion feasibility,
- learner-time plausibility.

## Anti-Pattern

Using generated word count as the primary measure of instructional richness, instructional quality, or page value.

Longer content is not automatically better learning.

Instructional quality should be evaluated using learner workload, activity completion feasibility, conceptual sufficiency, and learning-outcome attainment.
