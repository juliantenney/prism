## Generation depth (Class A hypotheses)

1. **GAM material body length ceiling** — All first-audit materials clustered &lt;80 words. *Update:* Iterations 1–7 + heteroscedasticity show longer bodies are achievable for strong archetypes; enzymes teaching materials remain compressed.

2. **Scenario richness floor** — Scenarios consistently lack stakes/constraints/data. Still worth testing; enzymes still showed placeholder “Case Alpha/Beta” patterns.

3. **Worked-example step depth** — Workshop Marx A1 produced adequate steps; self-study Marx did not. *Update:* Also distinguish **worked analysis/judgement** vs **process walkthrough** archetypes.

4. **DLA–GAM support ratio** — DLA specifies more than GAM teaches. Confirmed; Priority 1 packages target the GAM side.

5. **DP partial completeness** — knowledge_summary present; overview/purpose/study_tips absent. Retained as DP/Class issue — not archetype framework core.

6. **DP field emission** — Ed Psych DP emitted `learning_sequence` instead of `page_synthesis`. Retained for DP follow-on.

7. **Assessment item feedback** — First audit claimed zero feedback. **Correction:** field is often `explanation_or_rationale`; `feedback_display: answer_grid_end` affects visibility.

## Success criteria for future tests

When a new workflow is run, check whether the pattern:

- **Persists** → strengthens hypothesis for generation or brief-shape cause  
- **Varies with brief/profile** → note moderating factor  
- **Disappears under archetype package** → strengthens framework claim  
- **Varies only by domain with matched archetypes** → reopen secondary domain-bias investigation
