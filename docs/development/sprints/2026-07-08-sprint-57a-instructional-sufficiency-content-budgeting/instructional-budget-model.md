# Instructional Budget Model

This model treats learner workload as a composition of time categories, not a single text-volume proxy.

## Budget components

1. Page duration budget (total minutes)
2. Reading time
3. Thinking/planning time
4. Writing/production time
5. Discussion time
6. Application/analysis time
7. Reflection/review time
8. Optional extension time

## Modeling note

Word count contributes primarily to reading workload and sometimes production prompts.  
It does not directly represent cognitive effort, discussion load, or production cycles.

## Anti-Pattern

Using generated word count as the primary measure of instructional richness, instructional quality, or page value.

Longer content is not automatically better learning.

Instructional quality should be evaluated using learner workload, activity completion feasibility, conceptual sufficiency, and learning-outcome attainment.

## Open Question

Should future DLA outputs include an `estimated_learner_minutes` concept so that learner workload can be modeled directly rather than inferred from content volume?

This is a future-design consideration only.

## Example reminder

A task prompt such as "Write 150-200 words" can consume 15-25 minutes when read/think/plan/write/review phases are included.
