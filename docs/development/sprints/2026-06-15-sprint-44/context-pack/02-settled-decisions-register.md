## Ownership

### D-03 — Investigation-primary ownership

| Field | Content |
| ----- | ------- |
| **Decision** | For self-directed higher-education resources, the **governing inquiry** is the primary organising unit. Learner experience target: Question → Explanation → Investigation → Judgement → Reflection. Activities are phases within the investigation, not the definition of what the resource is. |
| **Evidence basis** | 43-02 ownership decision; Marx upstream inquiry arc; prototype blind review praised inquiry spine; Sprint 42 desired arc |
| **Status** | **Accepted** |
| **Reopen criteria** | User explicit rescope; or benchmark evidence that investigation-owned framing consistently fails learners vs resource-led design |

### D-04 — Resource-secondary ownership

| Field | Content |
| ----- | ------- |
| **Decision** | The learner experiences a **coherent self-study resource** that teaches while the investigation unfolds — orientation, explanation, conceptual development, synthesis — without replacing embedded activities. |
| **Evidence basis** | 43-02; Marx GAM exposition strength; 42-8 learning content as intellectual spine; prototype right column carries teaching substance |
| **Status** | **Accepted** |
| **Reopen criteria** | Evidence that resource voice should become primary owner over investigation for self-directed HE |
