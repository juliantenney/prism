# Sprint 30 shared vocabulary

| Term | Meaning |
|------|---------|
| **PEL** | Pedagogic Enrichment Layer — generation-layer enrichment programme |
| **PEC** | Pedagogic Enrichment Contract — named optional enrichment module |
| **Self-directed learner page** | `page_profile: learner`, `delivery_context: self_directed`, async study |
| **Learner-resource workflow** | Transcript/upload brief → self-study page (not classroom default) |
| **OUTPUT CONTRACT** | Auto-appended DLA/GAM prompt block for self-directed JSON shape |
| **Cognition pack** | Sprint 28 pack (`peer_instruction_pack`, …) — separate from PEC |
| **Passive renderer** | HTML export displays JSON fields only; no invented pedagogy |
| **Notebook-and-pencil** | Structured offline intellectual work — not chat tutoring |
| **Facilitated brief** | Workshop/seminar/peer instruction — PECs default off |
| **Intellectual coherence** | Cross-activity narrative sense — Sprint 30 Phase 1 target |

**E → O → G → C → R:** Brief → topology → generation → composition → render.
