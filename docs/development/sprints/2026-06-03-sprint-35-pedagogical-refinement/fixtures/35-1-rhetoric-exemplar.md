## learner_task (prefer)

```text
Complete both template tables: mark each statement Yes or No and write a one-sentence reason.
Using the scenario table, decide which arm has the narrower interval and whether they overlap. Quote both endpoints.
Complete the comparison table: write the purpose of each work and one defensible difference per row.
```

## learner_task (avoid)

```text
Learn about confidence intervals.
Explore Marx's ideas in the materials.
Understand the key concepts from the reading.
```

---

## expected_output (prefer)

```text
Both tables completed with defensible Yes/No judgements and brief reasons a peer could review.
A short comparison (3–5 sentences) that names both intervals, states narrow/wider and overlap, and quotes values from the table.
A filled comparison table a peer could review — each row has purpose text and one cited difference.
```

## expected_output (avoid)

```text
Understanding of confidence intervals.
Coverage of Marx's main ideas.
Notes on the topic.
```

---

## support_note (prefer)

```text
Focus on frequentist interpretation, not single-interval probability claims.
Quote intervals from the table; do not invent new endpoints.
Purpose is the author's aim, not a plot summary; difference must compare the two works.
```

## support_note (avoid)

```text
First read the introduction, then fill row 1, then ask your tutor if stuck.
The answer is Yes for the first statement because...
Discuss in pairs before submitting.
```

---

## Assessment stem (prefer)

```text
Given p-value = 0.03 at α = 0.05, work through:
1. What is your decision about H0?
2. What does the p-value represent in this test (one sentence)?
3. Would you call the result practically significant without extra context — yes or no, and why?
```

## Assessment stem (avoid)

```text
What is a p-value?
Define confidence interval.
Name the three levels.
```
